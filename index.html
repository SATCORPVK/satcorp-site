Measurement framework

We track leading indicators (capacity) and lagging indicators (outcomes), using shared definitions and privacy-aware aggregation.

  1. Define: outcomes, baselines, and equity markers.
  2. Collect: minimal data, maximum signal.
  3. Validate: partner review + sampling checks.
  4. Publish: methods, limitations, and code when possible.
